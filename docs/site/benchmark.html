Benchmark System

MHRQI is evaluated against classical state-of-the-art denoisers using multiple metric categories. All comparisons are performed on B-SCAN Retinal OCT images with multiplicative speckle noise.

Comparison Methods

BM3D

Block-Matching and 3D Filtering
Golden standard for Gaussian noise

NL-Means

Non-Local Means
Patch-based neighborhood filter

SRAD

Speckle Reducing Anisotropic Diffusion
Designed for ultrasound/OCT

Evaluation Metrics

Speckle Reduction

Metric Direction Description
SSI ↓ Lower Speckle Suppression Index (CoV ratio)
SMPI ↓ Lower Speckle Suppression & Mean Preservation Index
ENL ↑ Higher Equivalent Number of Looks (mean²/variance)
CNR ↑ Higher Contrast-to-Noise Ratio

Structural Preservation

Metric Direction Description
EPF ↑ Higher Edge Preservation Factor (wavelet-based)
EPI ↑ Higher Edge Preservation Index (Sobel gradient correlation)
OMQDI ↑ Higher Objective Measure of Quality of Denoised Images

# Single image comparison
python compare_to.py

# Statistical benchmark (9 images, Wilcoxon tests)
python statistical_benchmark.py

Statistical Testing

The statistical_benchmark.py script performs Wilcoxon signed-rank tests across 9 medical images to assess statistical significance.

Null hypothesis: No difference between MHRQI and competitor on a given metric.
Alternative: MHRQI performs differently (better or worse).
Significance level: α = 0.05
