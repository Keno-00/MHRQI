graph LR HCV["Position (HCV)"] --> Ancilla["Ancilla Target"] Ancilla -- "Controlled Flip" --> Target["Intensity Bits (0-255)"] Img["Pixel Intensity"] -- "Binary Map" --> Ancilla Target --> State["Basis-Encoded State"]

MHRQI-B uses Basis Encoding for pixel intensities. Each pixel's grayscale value is mapped to a binary integer and encoded across the intensity register using multi-controlled ancilla-driven bit flips. This ensures a compact, searchable representation of the image structure.
