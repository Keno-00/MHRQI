The framework supports multiple measurement-based and direct state simulation backends, including GPU-accelerated statevectors and memory-efficient Decision Diagrams for scaling to larger images.

06 // Image Retrieval System

graph LR Counts["Measurement Hits"] --> Prob["Hierarchical Probabilities"] Prob --> Interp["Intensity Interpolation"] Interp --> Recon["Final Image"]

Reconstruction involves sampling the hierarchical state to compute the **Hit-Rate** across position-intensity branches. In Basis-Encoded MHRQI, the intensity is reconstructed by aggregating bitwise measurement statistics from the multi-qubit intensity register.
