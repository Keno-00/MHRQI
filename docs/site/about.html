Project Context

This work represents an undergraduate thesis project exploring quantum computing approaches to image processing, with a focus on medical imaging (B-SCAN Retinal OCT scans). The project investigates hierarchical quantum image representations as an alternative to traditional flat encoding schemes.

Libraries and Tools

  • Quantum Computing: Qiskit, Qiskit Aer
  • Numerical Computing: NumPy, SciPy
  • Image Processing: OpenCV, scikit-image
  • Visualization: Matplotlib
  • Benchmarking: BM3D, BRISQUE, PIQE, PyWavelets (for OMQDI)
