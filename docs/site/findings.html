Summary of Findings

Strengths

  • Hierarchical encoding localizes pixel information, potentially reducing global state corruption from local perturbations
  • Comparable speckle reduction performance on OCT imagery for certain metrics (SSI, SMPI)
  • Outcome-weighted reconstruction preserves structural information while suppressing noise in homogeneous regions

Limitations

  • Simulation Only: All results are from classical simulation (Qiskit Aer). Not tested on real quantum hardware
  • Scalability: Circuit depth and qubit requirements scale with image size (2×L_max + 11 qubits; e.g., 27 for 256×256)
  • Performance Variability: Results vary across different noise types and image characteristics
  • Execution Time: Statevector simulation is computationally intensive for images larger than 256×256

Open Questions

  • How does this approach perform on real quantum hardware with noise and limited connectivity?
  • Can the consistency check be extended to multi-level hierarchy comparisons?
  • What is the optimal trade-off between qubit count and hierarchical depth?

This project represents an undergraduate thesis exploring quantum approaches to image processing. Results should be interpreted as preliminary findings from simulation-based experiments.
