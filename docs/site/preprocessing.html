Hierarchical Coordinate Vector (HCV)

The first step in the MHRQI pipeline is mapping classical Cartesian coordinates \((r, c)\) to a hierarchical path. This path encodes the pixel's position as a sequence of quadrant selections at each level of a quad-tree decomposition.

Mathematical Formulation

Hierarchy Depth

For an \(N \times N\) image with subdivision factor \(d=2\) (quad-tree):

\[ L_{\max} = \lfloor \log_d(N) \rfloor \]

Example: For a 256×256 image: \(L_{\max} = \log_2(256) = 8\) levels.

Subdivision Size

At each level \(k\), the subdivision size is:

\[ s_k = \frac{N}{d^k} \]

Quadrant Register Computation

At each level \(k\), compute the local quadrant indices \((q_y, q_x)\):

\[ q_y^{(k)} = \left\lfloor \frac{r \mod s_{k-1}}{s_k} \right\rfloor \] \[ q_x^{(k)} = \left\lfloor \frac{c \mod s_{k-1}}{s_k} \right\rfloor \]

HCV Construction

The complete Hierarchical Coordinate Vector is:

\[ \text{HCV}(r,c) = \left[ q_y^{(0)}, q_x^{(0)}, q_y^{(1)}, q_x^{(1)}, \ldots, q_y^{(L_{\max}-1)}, q_x^{(L_{\max}-1)} \right] \]

Each pair \((q_y^{(k)}, q_x^{(k)})\) maps to a pair of position qubits in the quantum circuit.

Worked Example

Given: Pixel at (r=3, c=3) in an 8×8 image

Step 1: Calculate L_max
L_max = log₂(8) = 3 levels

Step 2: Level 0 (s₀ = 8, s₁ = 4)
q_y⁽⁰⁾ = floor((3 mod 8) / 4) = floor(3/4) = 0
q_x⁽⁰⁾ = floor((3 mod 8) / 4) = floor(3/4) = 0

Step 3: Level 1 (s₁ = 4, s₂ = 2)
q_y⁽¹⁾ = floor((3 mod 4) / 2) = floor(3/2) = 1
q_x⁽¹⁾ = floor((3 mod 4) / 2) = floor(3/2) = 1

Step 4: Level 2 (s₂ = 2, s₃ = 1)
q_y⁽²⁾ = floor((3 mod 2) / 1) = floor(1/1) = 1
q_x⁽²⁾ = floor((3 mod 2) / 1) = floor(1/1) = 1

Result: HCV(3,3) = [0, 0, 1, 1, 1, 1]
Path: Top-left quadrant → Bottom-right sub-quadrant → Bottom-right pixel

Inverse Mapping (compose_rc)

To reconstruct coordinates from an HCV:

\[ r = \sum_{k=0}^{L_{\max}-1} q_y^{(k)} \cdot d^{L_{\max}-1-k} \] \[ c = \sum_{k=0}^{L_{\max}-1} q_x^{(k)} \cdot d^{L_{\max}-1-k} \]

Code: compose_rc(hcv, d=2) in utils.py
