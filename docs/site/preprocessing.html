The preprocessing stage transforms classical pixel coordinates into a Magnitude-Hierarchical Representation. Unlike standard quantum images (FRQI/NEQR) that use global position registers, MHRQI decomposes the image into nested quadrants (or qudits), where each level of the hierarchy represents a specific scale of spatial information.

02 // Preprocessing: The HCV Vector

The Hierarchical Coordinate Vector (HCV) is the core data structure of MHRQI. It maps a pixel's $(x,y)$ position to a sequence of local quadrant indices $Q_k$ across $L$ levels. This allows for scale-agnostic operations and efficient denoising through local diffusion.
