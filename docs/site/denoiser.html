The MHRQI denoiser implements Unitary Diffusion through partial Grover reflections. By coupling sibling blocks at each hierarchy level through quantum interference, the system effectively addresses boundary discontinuities (seams). Smooth regions are averaged within hierarchical sectors while structured regions (edges) are preserved via phase masking.
